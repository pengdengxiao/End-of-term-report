# 通知管理 Skill

## 技能描述
这是一个用于生成和管理通知的技能，可以根据用户输入生成格式规范的通知文档，自动添加部门信息，确保通知格式统一和规范。

## 功能说明
- 生成标准格式的通知
- 自动添加部门信息（如果用户未指定，则使用"xx部门"）
- 支持多种通知类型（会议通知、活动通知、公告通知等）
- 提供通知模板和格式建议

## 工具定义

### generate_notice 工具

**功能**：生成标准格式的通知

**参数**：
- `department` (可选): 部门名称，如果未提供则使用"xx部门"
- `title`: 通知标题
- `content`: 通知内容
- `notice_type`: 通知类型（meeting: 会议通知, activity: 活动通知, announcement: 公告通知, other: 其他）
- `effective_date` (可选): 生效日期，格式为YYYY-MM-DD
- `contact_person` (可选): 联系人
- `contact_phone` (可选): 联系电话

**返回**：
- 生成的完整通知内容

## 通知格式规范

### 基本格式
```
【部门】通知标题

通知内容

联系人：XXX
联系电话：XXX
日期：YYYY-MM-DD
```

### 会议通知格式
```
【部门】关于召开XXX会议的通知

各相关单位：

为了XXX，现决定召开XXX会议，具体事项通知如下：

一、会议时间：YYYY年MM月DD日 HH:MM
二、会议地点：XXX
三、参会人员：XXX
四、会议内容：XXX
五、其他事项：XXX

联系人：XXX
联系电话：XXX
部门名称
YYYY年MM月DD日
```

### 活动通知格式
```
【部门】关于举办XXX活动的通知

全体员工：

为了XXX，现决定举办XXX活动，具体事项通知如下：

一、活动时间：YYYY年MM月DD日 HH:MM
二、活动地点：XXX
三、参与人员：XXX
四、活动内容：XXX
五、报名方式：XXX
六、其他事项：XXX

联系人：XXX
联系电话：XXX
部门名称
YYYY年MM月DD日
```

### 公告通知格式
```
【部门】关于XXX的公告

全体员工：

XXX（公告内容）

特此公告。

部门名称
YYYY年MM月DD日
```

## 使用示例

### 示例1：生成会议通知
**输入**：
```json
{
  "toolcall": {
    "thought": "用户需要生成一个会议通知，包含会议时间、地点、参会人员等信息",
    "name": "generate_notice",
    "params": {
      "department": "人力资源部",
      "title": "关于召开2026年第二季度工作会议的通知",
      "content": "为了总结2026年第二季度工作情况，部署第三季度工作计划，现决定召开部门工作会议。会议时间：2026年7月15日 14:00，会议地点：公司会议室A，参会人员：全体部门成员，会议内容：1. 第二季度工作汇报 2. 第三季度工作计划 3. 团队建设讨论",
      "notice_type": "meeting",
      "effective_date": "2026-07-10",
      "contact_person": "张三",
      "contact_phone": "13800138000"
    }
  }
}
```

**输出**：
```
【人力资源部】关于召开2026年第二季度工作会议的通知

各相关单位：

为了总结2026年第二季度工作情况，部署第三季度工作计划，现决定召开部门工作会议，具体事项通知如下：

一、会议时间：2026年7月15日 14:00
二、会议地点：公司会议室A
三、参会人员：全体部门成员
四、会议内容：1. 第二季度工作汇报 2. 第三季度工作计划 3. 团队建设讨论

联系人：张三
联系电话：13800138000
人力资源部
2026年7月10日
```

```
五、报名方式：各部门负责人统一报名

联系电话：13900139000
xx部门
2026年7月20日
```

### 示例3：生成公告通知
**输入**：
```json
{
  "toolcall": {
    "thought": "用户需要生成一个公告通知",
    "name": "generate_notice",
    "params": {
      "department": "行政部",
      "title": "关于公司作息时间调整的公告",
      "content": "为了适应季节变化，提高工作效率，公司决定从2026年5月1日起调整作息时间。调整后的工作时间为：上午9:00-12:00，下午13:30-17:30。请全体员工知悉并遵守。",
      "notice_type": "announcement",
      "effective_date": "2026-04-20"
    }
  }
}
```

**输出**：
```
【行政部】关于公司作息时间调整的公告

全体员工：

为了适应季节变化，提高工作效率，公司决定从2026年5月1日起调整作息时间。调整后的工作时间为：上午9:00-12:00，下午13:30-17:30。请全体员工知悉并遵守。

特此公告。

行政部
2026年4月20日
```

## 注意事项
1. 部门名称如果未提供，将自动使用"xx部门"
2. 通知类型将影响生成的通知格式，请根据实际情况选择合适的类型
3. 生效日期格式应为YYYY-MM-DD，否则将使用当前日期
4. 联系人及联系电话为可选参数，可以根据实际需要添加
