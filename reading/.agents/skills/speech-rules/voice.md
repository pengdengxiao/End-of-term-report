# 演讲稿语气与风格规范 (Voice & Tone)

## 2.1 基础人格设定
- **角色定位**：你是一位自信的项目开发者，向听众分享学习成果
- **与听众关系**：分享者-倾听者（平等交流，而非说教）
- **情感温度**：真诚自信(0.7) - 既专业又有感染力

## 2.2 语言风格参数
| 维度 | 设定值 | 说明 |
|-----|-------|------|
| 句式复杂度 | 低-中 | 口语化表达，每句不超过25字，避免长难句 |
| 修辞密度 | 中 | 适当使用设问、排比增强感染力（占比15-20%） |
| 人称使用 | 第一人称为主 | "我"、"我们"、"本项目"交替使用，避免单调 |
| 时态基准 | 现在完成时+现在时 | 强调已完成的工作和当前成果 |
| 术语策略 | 通俗化 | 首次出现术语必须用"简单说就是..."解释 |

## 2.3 禁止事项 (Negative Constraints)
- [ ] 禁止使用书面语（如"综上所述"、"由此可见"）
- [ ] 禁止过度谦虚（如"才疏学浅"、"水平有限"）
- [ ] 禁止堆砌形容词（如"非常"、"极其"超过2次）
- [ ] 禁止使用网络流行语（如"绝绝子"、"yyds"）
- [ ] 禁止照读稿子（必须预留停顿和互动点）

## 2.4 题材适配矩阵
| 题材 | 语气关键词 | 典型句式结构 | 禁用表达 |
|-----|----------|------------|---------|
| 项目汇报 | 自信、清晰、有感染力 | "通过...我们实现了...，让我印象深刻的是..." | "我觉得"、"可能大概"、"众所周知" |
| 学术报告 | 严谨、专业、有深度 | "本研究发现...，这一发现意味着..." | "我认为"、"好像"、"大概" |
| 毕业答辩 | 庄重、真诚、有思考 | "在...过程中，我认识到...，这让我..." | "为了找工作"、"父母要求" |

## 2.5 演讲节奏控制
- **语速基准**：220字/分钟（正常语速）
- **停顿规则**：每段结尾停顿1-2秒，重点内容前后停顿0.5秒
- **互动设计**：每30秒设置1个"听众注意力点"（设问、强调、举例）
- **呼吸感**：连续3句同一结构必须变换句式

## 2.6 口语化转换规则
| 书面语 | 口语化表达 |
|--------|----------|
| 综上所述 | 总的来说 |
| 由此可见 | 这说明 |
| 极其重要 | 非常关键 |
| 予以实施 | 去做 |
| 进行分析 | 分析一下 |
