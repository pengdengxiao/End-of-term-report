Background 02
项目背景

为什么要做这个项目？

很多同学想学 AI Agent，但不知道从哪里开始。网上教程要么太复杂，要么太零散，很难形成完整的知识体系。

所以我想做一个真正适合初学者的教程，让大家能跟着做就能学会。

背景与目标 02
Achievements 03
项目成果

项目驱动的学习方式

整个教程围绕一个「智能工具助手 Agent」展开，从最基础的 LLM API 调用，到工具调用、记忆管理，最后整合成完整的 Agent 系统。

每个练习都对应一个核心模块，学完就能独立复现项目。

核心成果 03
Learning 04
个人收获

理解 AI Agent 的核心逻辑

它不是黑盒，而是由感知、记忆、规划执行这几个模块组成的。通过自己动手写代码，我真正明白了工具调用机制是怎么工作的，也学会了如何设计一个能自主决策的智能系统。

知识与技能 04
Challenges 05
遇到的挑战

克服困难，深化理解

过程中也遇到了不少挑战。比如如何让 Agent 记住历史对话，如何优化工具调用的效率。

但正是这些挑战，让我对 Agent 开发有了更深入的理解。

从使用者到开发者

这个项目让我从一个 AI Agent 的使用者，变成了一个能独立开发 Agent 的开发者。

我相信，只要掌握了这些核心原理，大家也能创建出更强大的 AI 应用。
